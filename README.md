## 关于 142vip

**142vip** 开源组织，口号是「积极拥抱变化」。

利用业余时间，结合工作经验，以 **JS / TS / Node / Vue 技术栈** 与 **计算机专业课（408）** 为主线，持续沉淀工程基建、学习笔记与实战项目；部分文章会选择性发表于微信公众号「储凡」。

当前主要在做三件事：

- **工程基建**：在 [`core-x`](https://github.com/142vip/core-x) 中沉淀可复用的 NPM 模块、CLI 与框架插件
- **知识沉淀**：维护 [408CSFamily](https://github.com/142vip/408CSFamily)、[JavaScriptCollection](https://github.com/142vip/JavaScriptCollection) 等学习与实战文档站
- **开源协作**：通过 Demo、文档与 Issue / PR，把工作中的最佳实践开放出来

## 开源代码

### Fairy-Cli

工程化 CLI 助手 [`@142vip/fairy-cli`](https://www.npmjs.com/package/@142vip/fairy-cli)（命令别名 `fa`），覆盖依赖安装、版本发布、CHANGELOG、代码规范、软著源码文档生成、镜像同步与部署等一站式能力，已发布至 NPM。

```shell
npm i -g @142vip/fairy-cli
fa -h
```

### NPM 模块

> 模块主要维护于 [`core-x`](https://github.com/142vip/core-x) 仓库，采用 **MonoRepo** 架构，[访问文档站点](https://142vip.github.io/core-x)
